# Family Expenses Manager

A simple React web app to manage family budgets and expenses by category. 

## Features

- Set a total family budget
- Add and remove customizable expense categories
- Assign budgets and track spent amounts per category
- Warnings when category or total expenses exceed budgets
- Data persistence using browser localStorage
